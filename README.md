# APOLLO CONNECTOR
Boilerplate for Apollo authentication/authorization

## Installation
from the project's root folder

`yarn install-all`

## Run whole development
from the project's root folder

`yarn start`

## Run a single server
You may also run servers separately

### VUE
from the project's root folder

`cd frontend && yarn dev`

### APOLLO
from the project's root folder

`cd backend && yarn launch`
